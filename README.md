# CarePoint

CarePoint është një ueb-faqe për një institucion shëndetësor (spital), e zhvilluar si projekt akademik në lëndën Web 1. Projekti paraqet informacion institucional, shërbime mjekësore dhe staf profesional përmes një ndërfaqeje të strukturuar dhe dizajni modern.

## Qëllimi i Projektit

Qëllimi i këtij projekti është zhvillimi i aftësive praktike në programimin e faqeve web nga ana e klientit, përmes ndërtimit të një ueb-faqeje për një institucion shëndetësor.

Projekti synon të demonstrojë njohuritë tona në përdorimin e HTML, CSS, JavaScript dhe React, si dhe aftësinë për të implementuar funksionalitete tipike të një ueb-faqeje institucionale, si prezantimi i shërbimeve mjekësore, stafit dhe informacioneve për përdoruesit.

Gjithashtu, projekti synon të zhvillojë aftësinë për menaxhimin e një projekti softuerik në GitHub, duke zbatuar praktikat më të mira të zhvillimit të software-it, përfshirë punën në grup, përdorimin e degëve (branches), commits dhe pull requests.

## Përshkrimi i Projektit

Ueb-faqja CarePoint përfshin disa faqe kryesore si: Ballina, Rreth CarePoint, Shërbimet mjekësore, Stafi dhe Kontakti. Fokusi kryesor është organizimi i përmbajtjes, navigimi i qartë dhe dizajni responsiv për pajisje të ndryshme.

Projekti është realizuar kryesisht si ueb-faqe statike, ndërsa ekziston edhe një version alternativ i implementuar në React.

## Teknologjitë e Përdorura

- HTML5
- CSS3
- JavaScript
- React

## Ekzekutimi i Projektit

### Versioni HTML / CSS / JavaScript
- Hapni skedarin `ballina.html` në një shfletues uebi
- Ose përdorni Live Server në Visual Studio Code

### Versioni React
1. Hyni në dosjen `carepoint-react`
2. Instaloni libraritë:
   ```bash
   npm install
   ```
3. Nisni serverin e zhvillimit:
   ```bash
   npm run dev
   ```
4. Hapni shfletuesin në:
   ```
   http://localhost:5173
   ```
